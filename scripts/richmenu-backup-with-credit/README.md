## วิธีกู้คืนเมนูนี้ในอนาคต (ถ้าต้องการกลับมาใช้ระบบเครดิต)

1. คัดลอกไฟล์ในโฟลเดอร์นี้ (`generate-richmenu.js`, `upload-richmenu.js`) กลับไปทับที่ `scripts/`
2. รัน `node scripts/generate-richmenu.js` เพื่อสร้าง `scripts/richmenu.png` ใหม่ (6 ปุ่ม รวม "เติมเครดิต")
3. รัน `node scripts/upload-richmenu.js` เพื่ออัปโหลดเป็นเมนูหลักบน LINE (จะลบเมนูเก่าที่ใช้อยู่ตอนนั้นทิ้งอัตโนมัติ)
